# Photo Portfolio Template

This is a template created for a friend. Created with a template release for others to view

## To set up
Uses npm, react + router dom, node.js
- Have all dependencies installed
- Create an assets folder and media subfolder to import media into
- Change into project folder
- Run site using "npm run dev"
